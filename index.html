使用帮助

👋 欢迎使用元答

元答是一个聚合多家大模型（通义千问、DeepSeek、智谱 GLM 等）的 AI 对话助手。直接在输入框输入问题即可开始对话。

🧠 思考过程

在「设置」中可开启「显示思考过程」。开启后，支持思维链的模型（如 GLM-5.2、Qwen3.5、DeepSeek 系列）会展示推理步骤，帮助你理解答案是如何得出的。部分模型不支持思维链，会自动降级为普通回答。

💬 每个模型历史对话独立

切换不同模型时，各自保留独立的对话记录，互不打扰。在左侧「历史对话」可查看与切换；新建对话会清空当前模型的聊天窗口。

🎭 对话角色

点击顶部「🎭 通用」可切换系统角色（写作助手、翻译专家、程序员、小红书文案、面试官等）。选定后，该角色会作为隐藏指令影响后续所有回答，让对话更有针对性。切换模型或新建对话会保留当前角色。

☁️ 个人云盘

通过左侧「个人云盘」可上传、管理文件（输入 ID + 密钥进入，每个 ID 独立且不可枚举）。密钥仅存于本机，换设备凭同一 ID + 密钥可进入同一云盘。

📋 模型调用日志

每次对话的模型调用（成功 / 失败 / 重试）都会记录在「设置 → 模型调用日志」中，便于排查问题。仅保留最近 50 条，存于本机。

⤓ 导出对话

点击顶部「⤓」按钮，可将当前模型的对话导出为 Markdown 文件，方便保存与分享。

⚠️ 关于「AI 生成 · 请核实信息」

本应用所有回答均由 AI 模型生成，可能存在事实错误、过时信息或虚构内容。请勿将其作为医疗、法律、金融等专业建议，重要信息请务必自行核实。你看到这句话，正是为了提醒你保持判断。

🤖 模型特点与推荐用法

  • Qwen3.5-27B：综合能力均衡，日常问答、写作首选。
  • DeepSeek-V3.2 / V4-Pro：代码、推理强，适合技术任务。
  • DeepSeek-R1：数学与逻辑推导，复杂问题友好。
  • GLM-5.2：支持思维链，专业 / 严谨场景推荐。
  • 通义万相 / Krea：文生图，输入描述出图。
  • 油价查询：查询实时油价，非聊天模型。

注：Kimi、腾讯混元、MiniMax 在当前 API 密钥下可能暂不可用，已自动降级保证不崩溃。
